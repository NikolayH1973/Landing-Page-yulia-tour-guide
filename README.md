## 🚀 Как открыть локально

Просто откройте `index.html` в любом браузере — достаточно двойного клика по файлу. Никакая сборка не нужна.

---

## ✏️ Как редактировать

Откройте `index.html` в любом текстовом редакторе (VS Code, Sublime, даже TextEdit) и найдите нужную секцию через `Ctrl+F` / `Cmd+F`.

### Изменить контактные данные

⚠️ **Контакты есть в трёх файлах** — `index.html`, `contact.html` и `yulia-hovich.vcf`. При смене обновите все три (Find & Replace в любом редакторе).

| Что | Найти |
|---|---|
| Номер WhatsApp | `972506887446` |
| Email | `hovich.yulia@gmail.com` |
| Instagram | `gid.israel.yulia` |

### Заменить фото Юли

Положите новое фото в папку `images/` под именем `yulia.jpeg` (заменив старое) — больше ничего менять не нужно. Если имя другое, поменяйте `<img src="images/yulia.jpeg" ...>` в секции About.

### Добавить фото в галерею

1. Сохраните новую картинку в папку `images/` (имя на английском, без пробелов): `images/my-photo.jpg`
2. Откройте `index.html`, найдите секцию `<!-- GALLERY -->` — там в комментарии готовые шаблоны для копирования (обычное / широкое / высокое фото). Не забудьте `images/` в `src=...`.

### Изменить тексты

Все тексты — внутри `index.html`. Просто найдите нужный фрагмент и замените. Тексты на русском, в чистом HTML — без специальной разметки.

## 📲 NFC-страница «Сохранить контакт»

Файл `contact.html` — отдельная мини-страница, оптимизированная под NFC-визитки и QR-коды. Сценарий: при встрече с туристом — он касается телефоном NFC-метки (или сканирует QR-код на визитке), открывается страница, одно нажатие — контакт сохранён в телефонной книге.

**Адрес страницы:** `yulia-israel.com/contact` (или `/contact.html`)

**Что на странице:**
- Фото Юли + имя + подзаголовок «Гид по Израилю»
- Большая кнопка «📲 Сохранить контакт» — скачивает `yulia-hovich.vcf`
- Кнопка «💬 Написать в WhatsApp» — открывает чат с предзаполненным «Здравствуйте, Юля!»
- Маленькая ссылка «Открыть сайт →» — ведёт на главную

**Если меняете контактные данные** — обновите ОБА файла: и `index.html`, и `yulia-hovich.vcf`. Тексты в `.vcf` редактируются в любом текстовом редакторе.

## 🌍 Деплой (как опубликовать в интернете)

Все варианты ниже — **бесплатные**.

### GitHub Pages
1. Загрузите проект на GitHub
2. Settings → Pages → Source: `main` branch / `/ (root)` → Save
3. Сайт будет доступен по адресу `https://USERNAME.github.io/REPO-NAME/`

### Netlify (самый быстрый)
1. Откройте [app.netlify.com/drop](https://app.netlify.com/drop)
2. Перетащите всю папку проекта в окно браузера
3. Готово — Netlify даст готовую ссылку

### Vercel
Из папки проекта в терминале:
```bash
npx vercel
```
