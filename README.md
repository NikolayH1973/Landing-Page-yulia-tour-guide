## 🚀 Как открыть локально

Просто откройте `index.html` в любом браузере — достаточно двойного клика по файлу. Никакая сборка не нужна.

## ✏️ Как редактировать

Откройте `index.html` в любом текстовом редакторе (VS Code, Sublime, даже TextEdit) и найдите нужную секцию через `Ctrl+F` / `Cmd+F`.

| Email | `hovich.yulia@gmail.com` | 2 |

### Заменить фото Юли

Положите новое фото в папку проекта под именем `yulia.jpeg` (заменив старое) — больше ничего менять не нужно. Если имя другое, поменяйте `<img src="yulia.jpeg" ...>` в секции About.

### Добавить фото в галерею

Откройте `index.html`, найдите секцию `<!-- GALLERY -->` — там в комментарии готовые шаблоны для копирования (обычное / широкое / высокое фото).

### Изменить тексты

Все тексты — внутри `index.html`. Просто найдите нужный фрагмент и замените. Тексты на русском, в чистом HTML — без специальной разметки.

## 🌍 Деплой (как опубликовать в интернете)

Все варианты ниже — **бесплатные**.

### GitHub Pages
1. Загрузите проект на GitHub
2. Settings → Pages → Source: `main` branch / `/ (root)` → Save
3. Сайт будет доступен по адресу `https://USERNAME.github.io/REPO-NAME/`

### Netlify (самый быстрый)
1. Откройте [app.netlify.com/drop](https://app.netlify.com/drop)
2. Перетащите всю папку проекта в окно браузера
3. Готово — Netlify даст готовую ссылку

### Vercel
Из папки проекта в терминале:
```bash
npx vercel
```

## 🎨 Что использовано

- Чистый HTML5, CSS3, vanilla JavaScript
- **Без** сборщиков, **без** npm, **без** внешних зависимостей
- Шрифты: Google Fonts (Playfair Display + Inter)
- Иконки: эмодзи + inline SVG
- Адаптивный дизайн, mobile-first
- Animation on scroll — IntersectionObserver
